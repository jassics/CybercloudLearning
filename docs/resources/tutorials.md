## Naming Convention

PDFs should follow this naming pattern:
```
Category-Topic-Description.pdf
```
Examples:
- `AppSec-OWASP-Top10-Guide.pdf`
- `Cloud-AWS-Security-Best-Practices.pdf`
- `DevSecOps-Pipeline-Security-Checklist.pdf`
